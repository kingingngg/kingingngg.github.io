二、易错成语理解

  • 红装素裹：只能用于形容雪后天晴，红日和白雪交相辉映的壮丽景色。
  • 相得益彰：只能形容两个事物之间互相帮助互相补充，更能显出各自的长处。(褒义词)
  • 断章取义：(贬义词)不顾全篇文章的内容，而只根据需要，孤立地截取其中一段或一句的意思。
  • 寻章摘句：搜寻、摘取文章的片段词句。指读书时仅局限于文字的推求。（贬义词）
  • 望文生义：不懂某一词句的正确意义，只从字面上去附会，做出错误的解释。
  • 附庸风雅：(多含贬义)为了装点门面而结交文人，参加有关文化活动。
  • 言不及义：(贬义词)只说些无聊的话，不涉及正经道理。
  • 鸠占鹊巢：(贬义词)比喻强占他人的居所。
  • 矫揉造作：(贬义词)形容过分做作，极不自然。
  • 间不容发：形容事物之间距离极小；形容与灾祸相距极近，情势极其危急。
  • 形销骨立：形容身体极其消瘦。
  • 如雷贯耳：形容人的名声很大。【振聋发聩：用语言文字唤醒糊涂的人】【震耳欲聋：形容声音很大。】
  • 箪食壶浆：用来形容军队受欢迎的情况。
  • 天伦之乐：指家庭中亲人团聚的快乐。（不能形容师生、同学、朋友共处的快乐）
  • 妙手回春：称赞医生医道高明，能把垂危的病人治好。（褒义词）
  • 高谈阔论：漫无边际地大发议论（多含贬义）。
  • 味同嚼蜡：形容写文章或说话枯燥无味。（注意使用对象）
  • 吹毛求疵：刻意挑剔毛病，寻找差错。（贬义词）
  • 强聒不舍：唠唠叨叨说个没完。
  • 李代桃僵：原比喻兄弟互相帮助，后借指以此代彼或代人受过。
  • 自吹自擂：(贬义词)自己吹喇叭，自己打鼓，比喻自我吹嘘。
  • 忸怩作态：(贬义词)故意做出不好意思或不大方的样子。
  • 一意孤行：(贬义词)不听劝告，固执地照自己的意思行事。
  • 富丽堂皇：(褒义词)形容场面、建筑或陈设等宏伟华丽、气势盛大；形容诗文词藻华丽。
  • 自惭形秽：泛指自愧不如别人。
  • 金戈铁马：借指威武雄壮的军队，也指战争或军旅生涯。
  • 抽丝剥茧：形容对事物分析细致且层次分明。

三、理解性默写

《行路难》李白
  1. 运用典故表现作者期待从政；含蓄表达渴望得到朝廷重用的心愿；引用两个典故，表示世路虽艰，但对前途并未丧失信心的诗句是: 闲来垂钓碧溪上，忽复乘舟梦日边。
  2. 表现诗人积极进取的精神、远大政治抱负、强烈自信心、实现人生理想的坚定信念：长风破浪会有时，直挂云帆济沧海。
  3. 通过动作细节刻画，揭示诗人内心苦闷抑郁之情：停杯投箸不能食，拔剑四顾心茫然。
  4. 象征人生路上的艰难险阻、壮志难酬；正面描写“行路难”的诗句是：欲渡黄河冰塞川，将登太行雪满天。
《酬乐天扬州初逢席上见赠》刘禹锡
  1. 诗中蕴含哲理；表明新事物必将取代旧事物；蕴含着新陈代谢的自然规律；表现了达观的人生态度；对世事的变迁和仕宦的沉浮表现出豁达的胸怀：沉舟侧畔千帆过，病树前头万木春。
  2. 用典故表达回归之后怀念故友、因世事变迁而怅惘心情；感叹物是人非：怀旧空吟闻笛赋，到乡翻似烂柯人。
  3. 写诗人长期被贬偏远之地，饱含无限辛酸与愤懑不平的诗句：巴山楚水凄凉地，二十三年弃置身。
  4. 既是对友人关怀的感谢，也是和之共勉，表现诗人坚定意志和乐观精神：今日听君歌一曲，暂凭杯酒长精神。
《水调歌头》苏轼
  1. 词中道尽了千载离人心愿，表达对天下人的美好祝愿的句子：但愿人长久，千里共婵娟。
  2. 阐释人生哲理,表现词人的旷达胸襟的句子是:人有悲欢离合，月有阴晴圆缺，此事古难全。
  3. 与“海上生明月,天涯共此时”意境相似的句子是:但愿人长久，千里共婵娟。
《月夜忆舍弟》杜甫
  1. 从听觉角度，描绘边塞秋天图景；渲染边塞的荒凉沉寂和浓重悲凉的气氛：戍鼓断人行，边秋一雁声。
  2. 通过写景，点明时令；浸润着作者主观感情；突出对故乡的感怀：露从今夜白，月是故乡明。
  3. 感叹离乱，亲人离散；概括出了安史之乱中人民饱经忧患的普遍遭遇的句子：有弟皆分散，无家问死生。
《长沙过贾谊宅》刘长卿
  1. 借贾谊遭贬暗寓了诗人自己迁谪的悲苦命运的诗句：三年谪宦此栖迟，万古惟留楚客悲。
  2. 渲染出贾谊故宅萧条冷落之景,抒写出寂寞怅惘的痛苦、无奈之情的诗句是:秋草独寻人去后，寒林空见日斜时。
  3. 用曲折的笔触暗讽当今皇上唐代宗昏聩无能；典故，含蓄地表达了自己对统治者的不满情绪；隐隐联系到自己沉沦坎坷是必然的句子是：汉文有道恩犹薄，湘水无情吊岂知。
  4. 描绘出一幅荒村日暮图，象征了当时国家的衰败局势，表现诗人对不合理现实的强烈控拆；是吊古伤今的悲叹：寂寂江山摇落处，怜君何事到天涯。

四、课内文言重要注释

《岳阳楼记》
  1. 谪：贬官
  2. 越明年：越：到。明年：第二年。
  3. 百废具兴：具：同“俱”，全、皆。
  4. 属予作文以记之：属：同“嘱”，嘱托。
  5. 胜状：胜景，美景。胜：美好。
  6. 前人之述备矣：备：详尽。☆
  7. 然则：如此，那么。
  8. 南极潇湘：极：至、到达。☆
  9. 迁：贬谪、降职。
  10. 骚人：泛指文人。
  11. 得无异乎：得无：表推测。
  12. 薄暮冥冥：薄：迫近。☆ 薄暮：傍晚。冥冥：昏暗。
  13. 登斯楼也：斯：这。
  14. 去国怀乡：去：离开。国：指国都。
  15. 忧谗畏讥：忧：担心。讥：批评指责。
  16. 感极而悲者矣：极：极点。
  17. 春和景明：景：日光。
  18. 翔集：时而飞翔，时而停歇。集：停息。
  19. 锦鳞：美丽的鱼。鳞：代指鱼。（只写鱼不得分）
  20. 或长烟一空：或：或许。一：全。
  21. 此乐何极：何极：哪有尽头。
  22. 宠辱偕忘：宠：光荣。辱：屈辱。偕：一起。☆
  23. 予尝求古仁人之心：尝：曾经。求：探求。☆
  24. 不以物喜：以：因为 ☆
  25. 先天下之忧而忧：先：在……之前。
  26. 微斯人：微：如果没有。
《醉翁亭记》
  1. 环滁皆山：环：环绕。
  2. 峰回路转：回：曲折、回环。
  3. 有亭翼然临于泉上者：翼然：像鸟张开翅膀一样。临：居高面下。
  4. 名之者谁：名：命名，取名。
  5. 饮少辄醉：辄：就。
  6. 醉翁之意：意：意趣，情趣。☆
  7. 得之心寓之酒也：得：领会。 寓：寄托。☆
  8. 野芳发而幽香：芳：花。 发：开放。☆
  9. 负者歌于途：负者：背着东西的人。
  10. 伛偻提携：伛偻：弯腰曲背，这里指老人。提携：牵扶，这里指被牵扶的人，即儿童。
  11. 泉香而酒洌：香：香甜。洌：清。
  12. 杂然而阵者：陈：陈列，摆开。☆
  13. 宴酣之乐：酣：尽兴地喝酒。
  14. 颓然乎其间者：颓然：倒下的样子。☆
  15. 树林阴翳：翳：遮盖。☆
  16. 太守之乐其乐也：乐：以……为快乐。
  17. 醒能述以文者：述：记述。以：用
  18. 太守谓谁：谓：为，是。☆
《湖心亭看雪》
  1. 人鸟声俱绝：绝：消失。
  2. 是日更定矣：是：这。更定：晚上八时左右。
  3. 余拏一小舟：拏：撑(船)。
  4. 拥毳衣炉火：拥：裹、围。毳：鸟兽的细毛。
  5. 沆砀：白汽弥漫的样子。
  6. 湖中焉得更有此人：焉得：哪能。更：还。
  7. 强饮三大白：强：竭力、尽力。白：古人罚酒时用的酒杯。
  8. 客此：客：客居。

五、课内文言翻译

《岳阳楼记》
  1. 此则岳阳楼之大观也，前人之述备矣。☆
    译文：这是岳阳楼的壮丽景象，前人的记述很详尽了。
  2. 微斯人，吾谁与归啊！☆
    译文：如果没有这种人，我同谁一道呢？
  3. 不以物喜，不以己悲，居庙堂之高则忧其民，处江湖之远则忧其君。☆
    译文：(是由于)不因外物和自己处境的变化而喜悲，在朝廷做官则为平民百姓忧虑，被贬谪到边远地区做地方官则替君主担忧。
  4. 而或长烟一空，皓月千里，浮光跃金，静影沉璧，渔歌互答，此乐何极！
    译文：偶尔或许大雾完全消散，皎洁的月光一泻千里，浮动的光像跳动的金子，静静的月影像沉入水中的玉璧，渔夫的歌声在你唱我和，这样的乐趣真是无穷无尽！
  5. 先天下之忧而忧，后天下之乐而乐。
    译文：在天下人忧之前先忧，在天下人乐之后才乐噫！
《醉翁亭记》
  1. 峰回路转，有亭翼然临于泉上者，醉翁亭也。☆
    翻译：山势回环，路也跟着转弯，有一座亭子，(亭角翘起)像鸟张开翅膀一样，高踞于泉水之上的，是醉翁亭啊。
  2. 醉翁之意不在酒，在乎山水之间也。山水之乐，得之心而寓之酒也。
    翻译：醉翁的意趣不在酒上，而在山光水色中啊。欣赏山水的乐趣，领会于心间，寄托在酒上。
  3. 醉能同其乐，醒能述以文者，太守也。☆
    翻译：醉了能够同大家一起欢乐，醒来能够用文章记述这事的人，是太守啊。
  4. 晦明变化者，山间之朝暮也。
    翻译：朝则自暗而明，暮则自明而暗，或暗或明，变化不一，就是山中的早晨和晚上。
《湖心亭看雪》
  1. 雾凇沆砀，天与云与山与水，上下一白。☆
    翻译：(湖上)冰花周围弥漫着白汽，天与云与山与水，浑然一体，白茫茫一片。
  2. 莫说相公痴，更有痴似相公者。☆
    翻译：不要说相公您痴，还有像您一样痴的人呢。
  3. 大雪三日，湖中人鸟声俱绝。☆
    翻译：接连下了三天的大雪，湖中行人、飞鸟的声音都消失了。
  4. 湖中焉得更有此人！
    翻译：“在湖中哪能还有这样的人呢！”

六、文言文内容理解

  1. 《岳阳楼记》
    通过一暗一明（一悲一喜）的写景对比，为下文议论做铺垫。表达出“不以物喜，不以己悲”的旷达胸襟和“先天下之忧而忧，后天下之乐而乐”的远大政治抱负。
  2. 《岳阳楼记》文章结尾“微斯人，吾谁与归”一句有何言外之意？
    答：①希望滕子京具有“古仁人之心”，不要因一时被贬而心灰意冷，应当志存高远，胸怀天下。其中含有对滕子京的劝慰之意。②含蓄地表达了作者希望与古仁人同道的想法，表达出“不以物喜，不以己悲”的旷达胸襟和“先天下之忧而忧，后天下之乐而乐”的远大政治抱负。
  3. 《醉翁亭记》
    表达了作者醉心山水、随遇而安的乐观精神和与民同乐的情怀。
  4. 《湖心亭看雪》
    表达了作者与众不同的闲情逸趣和孤傲清高、不随俗流的品性，寄托了作者对人生渺茫的感慨和对故国往事的感慨。

七、《艾青诗选》

考点1：作者介绍（常考选择题和填空题）
  • 艾青，原名蒋海澄，浙江金华人。中国现代诗人。
  • 1933年第一次用艾青的笔名发表长诗(成名作)《大堰河——我的保姆》。
  • 艾青诗歌的主题是爱国主义。
  • 20世纪30年代艾青诗歌的主要意象是“土地”和“太阳”。
  • 【“土地”意象象征多灾多难的祖国，凝聚着艾青对祖国和人民最深沉的爱。“太阳”意象象征着作者对于光明、理想、美好生活热烈不息的追求。】
  • 他的长诗《向太阳》《火把》借歌颂太阳、索求火把，表达了驱逐黑暗、坚持斗争、争取胜利的美好愿望，诗人也因此被称为“太阳与火把”的歌手。
考点2：诗歌中的意象解读（常考诗歌分析简答题）

中心意象：“土地”与“光明”(太阳)。

  1. “土地”的意象：象征生养自己又多灾多难的祖国，凝聚着诗人对祖国──大地母亲最深沉的爱。如《我爱这土地》《雪落在中国的土地上》《大堰河——我的保姆》。
  2. “太阳”的意象：表现了诗人对于光明、理想、美好生活热烈的不息的追求。如《向太阳》《黎明的通知》。

其他意象：

  • “礁石”：象征着坚强不屈的斗争精神。
  • “光”：赞颂“光”带来的社会文明，以及“像光一样坚强”的社会主义。
  • “镜子”：鞭挞和嘲讽了生活中的种种丑恶现象，弘扬了正直无私的品格。
  • “鱼化石”：象征着那些因社会不公而遭受苦难的知识分子。
  • “火把”：是时代的光明的象征。
  • “黎明”：象征着独立自由的曙光、革命的胜利、全国的解放。
  • “河流”：象征着郁结在人们心中的悲愤。
  • “风”：象征人民对侵略者的愤怒。
  • “启明星”：象征着驱逐黑暗而向往光明。
注：其余考点（两次创作高峰、艺术特色、“诗中有画”的特点、书籍推荐语、赏析角度、部分作品分析）可参照原文整理，格式与上述一致。

八、《水浒传》

基础信息
  • 【作者】：施耐庵（明末清初）。《水浒传》原名《江湖豪客传》《忠义水浒传》。
  • 【体裁】：是中国历史上第一部歌颂农民起义的章回体长篇白话小说。
  • 【内容主题】：记述了梁山好汉们从起义到兴盛再到最终失败的全过程，鲜明地表现了“官逼民反”的主题。
  • 【结构】：全书采用先分后合的链式结构，前四十回讲述单个英雄的故事，然后逐步发展到水泊梁山大聚义。第七十回后，写他们归顺朝廷，走向失败。
  • 【语言】采用古代白话，质朴生动，洗练明快，富有表现力。

九、议论文阅读

1. 议论文三要素

论点、论据、论证

2. 论点

（1）判断论点

论点是明确的判断，是表示肯定或否定的陈述语气的判断句，在形式上是个完整的句子。（但不能是疑问句，不能是一个短语，不能使用比喻的修辞。）一篇文章中的中心论点只有一个

（2）如何寻找论点

  1. 看题目（注意区分是论点还是论题）
  2. 看开头（注意是否有标志性语言会提示可能是论点）标志性语言：“所以”“可见”“总之”、“因此”、“总而言之”、“由此可见”“我确信”“我认为”“在我看来”“是……”
  3. 看中间（关注过渡句）
  4. 看结尾（会有总结句、点题句）
  5. 归纳整合分论点
  6. 根据论据反推论点
注：其余内容（论据的分类、论证方法的分类及作用、梳理论证思路、考点解析）可参照原文整理，格式与上述一致。

十、文学类文本阅读答题

※考点 1：情节概括题
  1. 看清题目要求
  2. 观察已给情节的答案要素
  3. 确定主语
  4. 字数相等，句式相同
※考点 2：体会加点词的表达效果

答题格式：

  1. ……词是……的意思，文中指……
  2. 生动形象地表现出……的……特点/生动传神地刻画了……的……情态
  3. 表现出人物……特征（分析人物形象）
  4. 表现出人物……心情
  5. 表达了作者……情感/品质

（若该词语有用修辞或描写方法要写出来）

注：其余考点（赏析句子的表达效果、理解某个句子、某个段落的作用、插叙的作用、理解标题的含义、标题的作用/好处、自然环境的作用、探究某一情节的作用、人称的作用）可参照原文整理，格式与上述一致。

十一、非连续性文本 考点及答题技巧

考点一：图表信息题
  1. 关注表格的标题，提取表头或表头的关键词作为答案的主语
  2. 数据看极端，横向对比/纵向对比；最高和最低
  3. 数据情况，答案不出现具体的数字，转化成文字

答案中常用词：“最”值：“最多”“最少”“绝大部分”“大多数”；数字呈现的趋势：“越来越……”“增加”“下降”；数字呈现比较关系：“A比B……”“较多”“占比最大”等

格式：对象+特点（变化趋势）

十二、一些题型做题原则

1. 做古诗“炼字”赏析题 总思路

  2. 答出写作手法（有就答出来）；
  3. 结合诗句内容；
  4. 点明作者情感和诗歌思想主旨。
2. 语言运用题

（1）拟写 活动邀请函、倡议书

  1. 格式正确，语言得体
  2. 要有尊称，“您”
  3. 活动主题、时间、地点要清楚
  4. 邀请语（诚挚地邀请您，期待您的莅临。）
  5. 落款：先写单位，再写日期。
